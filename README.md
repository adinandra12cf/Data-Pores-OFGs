# Data for Manuscript - Pores & OFGs

There are input files (.dat) and output files (.std) resulting from Geometry Optimization in OpenMX as follows:

- Only-Pores-CO2 > CO2 adsorbed on graphene with different pore diameters
- Only-OFGs-CO2 > CO2 adsorbed on graphene with different oxygen groups
- Small-Pores-OFGs-CO2 > CO2 adsorbed on small pore graphene with different oxygen groups
- Medium-Pores-OFGs-CO2 > CO2 adsorbed on medium pore graphene with different oxygen groups 
- Large-Pores-OFGs-CO2 > CO2 adsorbed on large pore graphene with different oxygen groups
